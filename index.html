Política de Privacidad

Qumran Watch es una Aplicación Web Progresiva (PWA) 100% frontend.

Recopilación de Datos

Esta aplicación NO recopila, almacena, ni transmite datos personales a ningún servidor externo.

Toda la información procesada (coordenadas GPS, preferencias de visualización) se almacena exclusivamente en el almacenamiento local del navegador (localStorage) y nunca abandona el dispositivo del usuario.

Geolocalización

El acceso a la ubicación GPS se utiliza únicamente para calcular la salida y puesta del sol. Esta información se almacena localmente y no se comparte con terceros. El usuario puede denegar el permiso de geolocalización; en ese caso, se usará Jerusalén (31.7683, 35.2137) como valor por defecto.

Servicios de Terceros

Esta aplicación no integra scripts de terceros, analytics, trackers ni publicidad.

Cookies

No se utilizan cookies de ningún tipo.
